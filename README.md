# Bilal Hassan — Portfolio

Static single-page portfolio site. No build step required.

## Structure
- `index.html` — the site
- `assets/` — project screenshots

## Local preview
Just open `index.html` in a browser, or serve it:
```
python3 -m http.server 8000
```

## Deploy
Static hosting (Vercel, Netlify, GitHub Pages) — no build command needed, root as publish directory.
